# AbuZhang Restaurant playground (React)

A simple web application built with **React** that allows users to manage a restaurant menu in an interactive way.

---

##  Features

-  Add new dishes with:
  - Dish name
  - Dish description
  - Category selection
-  Delete dishes easily
-  Organized menu categories:
  - Main Dishes (Heavy meals)
  - Snacks
  - Desserts
-  Comment section where users can add comments
-  Clean component-based structure using React

---

## Built With

- **React**
- **JavaScript (ES6)**
- **HTML5**
- **CSS3**

---

## Purpose

This project was built as a **hands-on pproject** after completing a React course that was mostly theoretical.  
The main goal was to apply core React concepts in a real, practical project.
